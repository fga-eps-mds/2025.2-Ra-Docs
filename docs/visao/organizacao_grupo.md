## Organização em Grupo

A organização em grupo é fundamental para garantir a colaboração e a eficiência no desenvolvimento de software. No contexto do nosso projeto, adotamos uma estrutura que favorece a comunicação e a integração entre os membros da equipe.

### Comunicação e Colaboração

Utilizamos ferramentas de comunicação como *Whatsapp* e *Discord* para facilitar a troca de informações entre os grupos. Reuniões regulares são agendadas para discutir o progresso, alinhar expectativas e resolver problemas.

### Controle de Versão
Utilizamos o GitHub para controle de versão, permitindo que todos os membros do grupo possam contribuir de forma organizada e rastreável. Cada membro é responsável por criar branches para suas tarefas e seguir as diretrizes de commits estabelecidas no guia de contribuição.
