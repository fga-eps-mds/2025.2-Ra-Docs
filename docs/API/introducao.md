Esta é a documentação oficial da API para o projeto Aton!

Bem-vindo(a) à API do Aton. Esta documentação fornece todos os detalhes necessários para interagir com os recursos do nosso backend.

O Aton é um aplicativo móvel desenvolvido para solucionar a descentralização das informações sobre eventos esportivos na Universidade de Brasília (UnB). Nosso objetivo é centralizar a divulgação de eventos, treinos e partidas promovidos por atléticas e equipes amadoras, fomentando a integração da comunidade acadêmica, o lazer e a prática de esportes.

Esta API é a espinha dorsal do aplicativo Aton, fornecendo os endpoints necessários para gerenciar usuários, autenticação, grupos (atléticas e times), postagens de eventos, partidas e notificações.

Através desta API, você poderá:

* Gerenciar a autenticação e os perfis de usuários (alunos, torcedores, atléticas).
* Criar, consultar, atualizar e deletar grupos, times amadores e atléticas.
* Publicar e consumir informações sobre eventos e partidas.
* Gerenciar interações sociais, como seguir grupos e postar em feeds.
